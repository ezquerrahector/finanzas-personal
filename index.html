🧠 IA real con Firebase Functions
La IA ya no usa API key en el navegador. Ahora llama a una función segura de Firebase:

Sube la carpeta functions incluida en la v3.4 y ejecuta:
firebase deploy --only functions

Tu API key vive solo en Firebase Functions, no en el HTML.
