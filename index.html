🧠 IA real con Firebase Functions
La IA usa Firebase Functions y los tickets se guardan en Firebase Storage:
analyzeFinance

Antes de desplegar, configura el secret de OpenAI y sube la carpeta functions:
firebase functions:secrets:set OPENAI_API_KEY
firebase deploy --only functions

Tu API key vive solo en Firebase Functions, no en el HTML.
